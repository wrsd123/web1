Definition of HTML Abbreviation of HTML is "Hypertext Markup Language". HTML expresses structural meaning that shows main text like title, paragraph, list as well as provides methods that can make structural document. HTML also can be used to save object and image and make conversational form.

HTML is written by HTML elements that is tag surounded with "<>" in contents of webpage. HTML can call or include Script like CSS which definite the exterior and arrangement, JavaScript that influences the behavior of HTML processing unit like web browser.
